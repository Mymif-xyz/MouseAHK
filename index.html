How to use your script
  1. Download and install AutoHotkey v2 from autohotkey.com
  2. Click Export .ahk in Mouse AHK to save your script file
  3. Double-click the .ahk file to run it
  4. A green H icon appears in the system tray, the script is running
  5. To stop it: right-click the tray icon → Exit
