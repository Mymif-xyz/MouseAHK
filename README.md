# MouseAHK
Mouse AHK is a browser-based tool for generating AutoHotKey scripts with custom macros, shortcuts, and gestures, no coding or AHK experience required ◝(ᵔᵕᵔ)◜

# What can it do?
 - App Profiles – Create separate setups for different applications
 - Macro Builder – An easy, visual way to build macros without writing code
 - Import/Export – Move your setup between devices or back it up easily
 - Keyboard Support – Despite the name it also supports keyboards!
 
# How do I start?
1. Visit https://mymif-xyz.github.io/MouseAHK/
2. Follow the on-screen tutorial
3. Export and run with AutoHotkey

The site also includes tutorials for installing AHK, running/stopping your script, and setting it to launch automatically on startup.

# Privacy
MouseAHK is fully open source and all your data is stored locally. The final script is completely independent — no logins, accounts, or internet connection required to use them.
